STEDT Database Now Online!

  • 2011.01.01 We are pleased to announce that the database is now available for public use via the world wide web. The database contains (as of this announcement) 432,976 lexical items from 566 languages, protolanguages, and dialects of the Sino-Tibetan linguistic area.

    An initial version of the web interface may be found at:

    http://stedt.berkeley.edu/search
